# Friday - AI Virtual Assistant
Your personal AI companion built with HTML, CSS & JS.

This repository contains a browser-only virtual assistant demo that uses the Web Speech API for voice recognition and speech synthesis to respond to voice commands. A simple UI with quick commands, settings, and a visual assistant face is included for interaction.

## Features
- Voice recognition (Web Speech API)
- Text-to-Speech (Speech Synthesis)
- Configurable settings: language, voice gender, speech rate, wake word, and theme
- Quick Commands: time, date, weather, jokes, calculator
- Responsive UI with animated assistant face and status display

## How to run
1. Open `index.html` in a modern browser (Chrome, Edge) that supports Web Speech API.
2. Click "Wake Friday" to activate the assistant, then click "Voice Command" and speak.
3. Use the Settings panel to change language and voice options.

## Development
- Edit `index.html`, `style.css`, and `script.js`.
- For best dev experience, use Live Server in VS Code or any static HTTP server.
